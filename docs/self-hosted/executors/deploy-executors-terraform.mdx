# Deploying Sourcegraph executors using Terraform

[Terraform modules](https://learn.hashicorp.com/tutorials/terraform/module-use?in=terraform/modules) are provided to
provision machines running executors on AWS and Google Cloud. Choose your cloud provider:

-   **AWS** — [Deployment guide](/self-hosted/executors/deploy-executors-terraform-aws) · [Module source](https://github.com/sourcegraph/terraform-aws-executors)
-   **Google Cloud** — [Deployment guide](/self-hosted/executors/deploy-executors-terraform-gcp) · [Module source](https://github.com/sourcegraph/terraform-google-executors)

## Terraform Version

The executor Terraform modules require Terraform `>= 1.1.0, < 2.0.0`.
