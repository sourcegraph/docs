# Deploying Sourcegraph executors using Terraform

[Terraform modules](https://learn.hashicorp.com/tutorials/terraform/module-use?in=terraform/modules) are provided to
provision machines running executors on AWS and Google Cloud. Choose your cloud provider:

-   [AWS](/self-hosted/executors/deploy-executors-terraform-aws) — using the [`sourcegraph/executors/aws`](https://github.com/sourcegraph/terraform-aws-executors) module
-   [Google Cloud](/self-hosted/executors/deploy-executors-terraform-gcp) — using the [`sourcegraph/executors/google`](https://github.com/sourcegraph/terraform-google-executors) module

## Terraform Version

The executor Terraform modules require Terraform `>= 1.1.0, < 2.0.0`.
