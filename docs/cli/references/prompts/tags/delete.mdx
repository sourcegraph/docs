# src prompts tags delete

Delete a tag from your Sourcegraph instance.

## Usage

```bash
src prompts tags delete <tag-id>
```

## Examples

```bash
# Delete a tag by ID
src prompts tags delete UHJvbXB0VGFnOjE=

# The command will confirm successful deletion
src prompts tags delete UHJvbXB0VGFnOjE=
# Output: Prompt tag deleted successfully.
```
