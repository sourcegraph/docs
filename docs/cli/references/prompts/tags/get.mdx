# src prompts tags get

Get details about a specific tag by name.

## Usage

```bash
src prompts tags get <tag-name>
```

## Examples

```bash
# Get tag by name
src prompts tags get go

# Get tag details for machine learning
src prompts tags get machine-learning
```
